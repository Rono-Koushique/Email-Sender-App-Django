**** Info ****

1. Email sending system for webpage using django
2. The app ('Sender') can be integrated to any django app
3. Currently the email can be received locally to the django console
4. Frontend is created using HTML and bootstrap


**** Important note for re-using ****

1. Copy the 'Sender' app to your project
2. Add the 'app'config file to your project settings in "installed app" section
3. Add the email_backend object at the end of the project settings
4. For practical implimentation search online for django implementation of your required email sending service
